Description

Implement the shadow-write path specified in ADR-0042. Writes go to both Redis and Postgres; reads continue from Redis. A metric (session.shadow.diff) compares the two and alerts on divergence > 0.1%.

This is intentionally a one-way valve for the first two weeks — no read traffic touches Postgres until divergence sits at zero for 72h.
