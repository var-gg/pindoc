# Pindoc Assets

This directory holds public-facing README and launch assets.

Use stable, descriptive filenames:

- `pindoc-public-demo-reader.png`
- `pindoc-public-demo-today.png`
- `pindoc-public-demo-search.png`

Before committing an asset, check that it does not show tokens, private emails,
local usernames, internal hostnames, private repo names, or unreleased domains.
Prefer one curated screenshot plus a live demo link over GIF/MP4 for the first
public README.
